## Puesta en marcha

1. Clonar repositorio
2. Ejecutar comando para restablecer los _node_modules_
   1. ```bash
      pnpm install
      ```
3. Poner en marcha la **app** con el comando:
   1. ```bash
      pnpm run dev
      ```
